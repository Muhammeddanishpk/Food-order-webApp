## Run Locally

Clone the project

```bash
  git clone https://github.com/Muhammeddanishpk/Food-order-webApp.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
